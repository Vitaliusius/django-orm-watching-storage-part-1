# Пульт охраны банка

Это внутренний репозиторий для сотрудников банка "Сияние". Если вы попали в этот репозиторий случайно, товы не сможете его запустить, т.к. у вас нет доступа к БД, но можете свободно использовать код верстки или посмотреть как реализованы запросы к БД.

Пульт охраны - это сайт, который можно подключить к удаленной базе данных с визитами и карточками пропуска сотрудников нашего банка.

# Как установить

1. Вы можете клонировать общедоступный репозиторий:

```
git clone https://github.com/devmanorg/django-orm-watching-storage-part-1
```

2. Затем, если требуется создать виртуальное окружение:

```
python -m venv имя_окружения
```

Запускаем окружение:

```
имя_окружения\Scripts\activate.bat
```


3. Python 3.14.2 должен быть уже установлен. Затем используйте ```pip``` (или ```pip3```, есть конфликт с Python2) для установки зависимостей:

```
pip install -r requirements.txt
```


Переменные окружения: DB_HOST(хост), DB_PASSWORD(пароль), SECRET_KEY(секретный ключ), DB_PORT(номер порта), DB_NAME(имя), DB_USER(пользователь) . Это переменные в которых должны храниться данные для подачи запроса в базу данных. Для работы программы необходимо создать файл ".env"  и просвоить переменным соответствующие значения:
```
SECRET_KEY = '...'

DB_PASSWORD = '...'

DB_HOST = '...'

DB_PORT = '...'

DB_NAME = '...'

DB_USER = '...' 
```

4. Затем запускаем программу из каталога репозитория:

```
main.py
```

И видим в консоли (пример):

```
Всего пропусков:  100
Активных пропусков:  89 
```
